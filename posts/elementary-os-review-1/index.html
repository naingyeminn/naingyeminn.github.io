Elementary OS - Review (1)

April 11 2014
elementaryos review

လက်ရှိ Laptop မှာ Ubuntu 12.04 LTS ပဲသုံးနေခဲ့တာ တော်တော်ကြာနေပါပြီ။ ကြားထဲမှာ 12.10, 13.04 နဲ့ 13.10 version သုံးခုတောင်ထွက်ခဲ့သည့်တိုင်အောင်မပြောင်းသေးပဲ Stable ဖြစ်တဲ့ LTS ကိုပဲဆက်သုံးဖြစ်နေတာပါ။ အခု လာမယ့် April ဆိုရင် နောက်ထပ် LTS version အသစ်တစ်ခုဖြစ်တဲ့ 14.04 ထပ်ထွက်တော့မှာဖြစ်တဲ့အတွက် version အသစ်ကိုမပြောင်းခင် တခြား distro လေးတွေပြောင်းစမ်းဖို့စိတ်ကူးမိတာနဲ့ပဲ နောက်ဆုံး ဒီ Elementary OS လေး စမ်းကြည့်ဖို့ ရွေးလိုက်ပါတယ်။

Elementary OS က ubuntu-based distro တစ်ခုပါပဲ။ Ubuntu ကိုအခြေခံပြီး မွမ်းမံထားတဲ့ OS တစ်ခုပါ။ သူ့ကို eOS လို့ အတိုကောက်ခေါ်ကြလေ့ရှိပါတယ်။ အခုစမ်းကြည့်ရသလောက်တော့ တော်တော်လေးကို ရိုးရှင်းပြီး အသုံးပြုရလွယ်ကူတာတွေ့ရပါတယ်။ သုံးကြည့်ရသလောက်တော့ အပြည့်အဝပြည့်စုံကောင်းမွန်တဲ့ OS တစ်ခုလို့မဆိုနိုငေ်ပမယ့် အတိုင်းအတာတစ်ခုထိ စိတ်ကြေနပ်စရာကောင်းတဲ့ OS လေးတစ်ခုဖြစ်တယ်လို့တော့ဆိုချင်ပါတယ်။ ထူးခြားချက်လေးတွေ၊ အားသာချက်လေးတွေရှိသလို အားနည်းချက်လေးတွေလဲရှိပါတယ်။

User Interface အနေနဲ့ တော်တော်လေးကိုရိုးရှင်းပြီး သပ်ရပ်ပါတယ်။ Pantheon ဆိုတဲ့ Desktop Environment ကိုသုံးထားတာပါ။ Window, Menu, dialog box ပုံစံတွေ ကတော့ Gnome3 နဲ့ဆင်တူပါတယ်။ File Manager အနေနဲ့ကတော့ Ubuntu မှာလို Nautilus မဟုတ်တော့ပဲ Pantheon-Files ဖြစ်သွားပါတယ်။ Nautilus ကိုယှဉ်နိုင်လောက်အောင် feature မစုံသေးပေမယ့် အသုံးပြုရတာတော့ရိုးရှင်းလွယ်ကူပါတယ်။ ဒါပေမယ့် Nautilus မှာလို Folder Share ဖို့ Sharing Options မပါပါဘူး။ ဒါကြောင့် Network ကနေ File တွေ Share လုပ်ချင်တဲ့အခါ Samba သွင်းပြီး manually configure လုပ်ပေးရပါတယ်။ အဲလိုမှမဟုတ်ရင်တော့ Nautilus ထပ်တင်ပြီးသုံးလို့ရပါတယ်။ Installation ပြီးပြီးချင်းမှာ ပြဿနာရှိတာက Power Consumption ပါ။ Desktop တစ်လုံးမှာ တင်ပြီးသုံးမယ်ဆိုရင်ပြဿနာမရှိပေမယ့် Laptop အနေနဲ့ကတော့ အားနည်းချက်တစ်ခုလို့ဆိုရမှာပါ။ Ubuntu မှာ ၅နာရီဝန်းကျင်လောက် Battery နဲ့သုံးလို့ရပေမယ့် သူမှာတော့ ၂နာရီခွဲ သုံးနာရီလောက်ပဲသုံးလို့ရပါတယ်။ ဒါပေမယ့် application တချို့သုံးပြီး tweak လုပ်လိုက်မယ်ဆိုရင်တော့ ၎နာရီ ၎နာရီခွဲလောက်သုံးလို့ရသွားမှာပါ။

နောက်တစ်ချက်က Application ပိုင်းပါ။ သူ့မှာ Ubuntu လို Office Suite ပါမလာပါဘူး။ LibreOffice ပြန်တင်ပေးရပါတယ်။ Email Client အနေနဲ့ကတော့ Geary Mail ဆိုတဲ့ application လေးပါပါတယ်။ ဒါပေမယ့် သူ့မှာက IMAP နဲ့ SMTP ပဲ support လုပ်တာပါ။ POP3 သုံးလို့မရပါဘူး။ POP3 သုံးချင်ရင်တော့ Thunderbird၊ ဒါမှမဟုတ် Evolution တစ်ခုခုထပ်တင်ပြီးသုံးရပါလိမ့်မယ်။ Default Web Browser ကတော့ Midori ပါ။ သုံးရအဆင်ပြေတယ်ဆိုပေမယ့် Firefox တို့ Chrome တို့မှာလို Plug-ins စုံစုံလင်လင်မရှိတဲ့အတွက် သုံးနေကျ Browser တစ်ခုခု ထပ်တင်တာပိုကောင်းပါလိမ့်မယ်။

Ubuntu နဲ့ ကွာခြားတာက သူ့မှာ Ubuntu လို Application စုံစုံလင်လင်မပါတာပါပဲ။ Disk Utility, System Monitor, Remote Desktop Client, Transmission BitTorrent Client တို့လို ရှိသင့်တဲ့ Application တွေမပါပါဘူး။ ထပ်တင်ဖို့လိုပါတယ်။ အဲဒီအတွက် အသေးစိပ် ဆက်လကေ်ဖာ်ပြပေးပါဦးမယ်။

Coloring Log Files

October 14 2015
linux server administration tips&tricks ccze log

Linux Administrator တွေအနေနဲ့ Log Analyzer တွေမျိုးစုံရှိနေပေမယ့် တစ်ခါတစ်ရံ လိုအပ်လို့ Log File တွေကိုမွှေနှောက်ရမယ်ဆိုလျှင် အာရုံနောက်ကြမယ်ထင်ပါတယ်။ Log File တွေရဲ့ထုံးစံအတိုင်း Text တွေကို ပြွပ်သိပ်နေအောင်ရိုက်ထုတ်ထားတာဖြစ်တဲ့အတွက် ဖတ်ရတဲ့သူအတွက် ဘယ်အပိုင်းက ဘာဆိုတာကို မနည်း အာရုံစိုက်ခွဲခြားနေရပါလိမ့်မယ်။ အဲဒီပြဿနာကိုတော့ CCZE ကကူညီဖြေရှင်းပေးနိုင်ပါတယ်။ CCZE ဆိုတာကတော့ command-line tool လေးတစ်ခုပဲဖြစ်ပါတယ်။ Linux Admin တွေအနေနဲ့ server ကို ssh နဲ့ access လုပ်နေတဲ့အချိန် Log File တွေကိုဖတ်ရှုဖို့လိုလာခဲ့မယ်ဆိုလျှင်တော့ ဒီ Tool လေးက အရမ်းအသုံးတည့်ပါတယ်။ Log ကိုသက်ဆိုင်ရာ အပိုင်းအခြားအလိုက် အရောင်ခွဲခြားပြီး ပြသပေးတာဖြစ်တဲ့အတွက် ဖတ်ရှုသူအနေနဲ့ မြန်ဆန်လွယ်ကူစွာ နားလည်သဘောပေါက်နိုင်မှာပါ။

Debian/Ubuntu မှာတော့ -

$ sudo apt-get install -y ccze

ဆိုပြီး Install တန်းလုပ်လို့ရပါတယ်။

RHEL/CentOS/Fedora စတဲ့ Distro တွေမှာတော့ EPEL ကနေ Install ပြုလုပ်နိုင်ပါတယ်။

$ sudo yum install -y ccze

Install ပြုလုပ်ပြီးပြီဆိုလျှင်တော့ စသုံးလို့ရပါပြီ။ Command-line Tool ဆိုတဲ့အတိုင်း command တစ်ခုခုကနေထွက်လာတဲ့ output ကို pipe | သုံးပြီး ccze ဆီကို ပို့ပေးလိုက်ရုံပါပဲ။ နမူနာအနေနဲ့ CentOS 7 ပေါ်မှာ အသုံးပြုပြပါမယ်။ အောက်ပါ Command နှစ်ကြောင်းကို Terminal မှာရိုက်ကြည့်လိုက်ပါ။

$ sudo tail -3 /var/log/messages
$ sudo tail -3 /var/log/messages | ccze -A

ပုံမှာမြင်တွေ့ရတဲ့အတိုင်း Color ပါတာနဲ့ မပါတာကို ယှဉ်ပြီးတွေ့ရပါလိမ့်မယ်။

နောက်ပြီး Log Output ကို HTML ဖိုင်အနေနဲ့လည်း ထုတ်ယူလို့ရပါသေးတယ်။

$ sudo tail -30 /var/log/messages | ccze -h > ~/Documents/msg.html

Output ကိုတော့ User ရဲ့ Documents folder အောက်ထဲမှာ msg.html ဆိုပြီး HTML file အနေနဲ့တွေ့ရပါလိမ့်မယ်။ Double click နှိပ်ပြီး ဖွင့်ကြည့်လိုက်မယ်ဆိုလျှင်တော့ ပုံမှပြထားတဲ့အတိုင်း Browser မှာမြင်တွေ့ရပါလိမ့်မယ်။

ပုံထဲမှာတော့ Default Color Scheme အတိုင်းဖြစ်နေတာရယ်၊ VM ပေါ်မှာဖြစ်တဲ့အတွက် Graphic Resolution နည်းနေတာရယ်တို့ကြောင့် ကြည့်ရတာတော့သိပ်အဆင်မပြေပါဘူး။ Default Color ကိုပြင်ချင်တယ်ဆိုလျှင်တော့ Config File /etc/cczerc ထဲမှာ ဝင်ပြင်လို့ရပါတယ်။ HTML output ရဲ့ Background Color ကိုပြောင်းမယ်ဆိုလျှင်တော့ /etc/cczerc ရဲ့ အောက်ဆုံး line number 93 မှာရှိတဲ့ cssbody #404040 ကိုပြင်ရုံပါပဲ။ အခြား Color တွေကိုလည်း စိတ်ကြိုက်ပြင်ဆင်နိုင်ပါတယ်။

အပေါ်ပုံကတော့ cssbody #000000 ပြောင်းပြီး output ထုတ်ထားတဲ့ HTML ဖိုင်ကို Safari Browser ပေါ်မှာမြင်တွေ့ရတာပဲဖြစ်ပါတယ်။

CCZE မှာ နောက်ထပ် Option တွေအများကြီးရှိပါသေးတယ်။ အသေးစိပ်ကိုတော့ man page သုံးပြီးလေ့လာနိုင်ပါတယ်။

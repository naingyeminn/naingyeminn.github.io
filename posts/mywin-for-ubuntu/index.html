MyWin for Ubuntu

July 31 2016
unicode keyboard input mywin keyman ubuntu

Ubuntu User တွေအနေနဲ့ MyWin ကို သုံးချင်တယ်ဆိုရင် သာမန် User တစ်ယောက်အတွက် Install လုပ်ရတာ အနည်းငယ်ခက်ခဲနိုင်တဲ့အတွက် mm-kb လိုပဲ အလွယ်ကူအောင် Makefile လေးနဲ့ packge ထုတ်ပေးထားပါတယ်။

ဘာမှထွေထွေထူးထူးလုပ်နေစရာမလိုပါဘူး။ Terminal ဖွင့်ပြီး ဖေါ်ပြထားတဲ့ Command တွေကို တစ်ကြောင်းချင်းစီရိုက်သွားရံုပါပဲ။

$ wget https://github.com/naingyeminn/mywin-ubuntu/archive/master.zip -O mywin-ubuntu.zip
$ unzip mywin-ubuntu.zip
$ cd mywin-ubuntu-master
$ sudo make install
$ ibus-daemon -rdx
$ im-config -n ibus
$ gsettings set org.freedesktop.ibus.panel show 0

ပြီးလျှင်တော့ ပုံမှာပြထားတဲ့အတိုင်း Text Entry Settings… ထဲဝင်လိုက်ပါ။

”+” button ကိုနှိပ်ပြီး Input Source list ထဲ “my-win” ကိုရှာပြီး Add လုပ်လိုက်ပါ။

Input sources to use ဆိုတဲ့ list ထဲမှာထပ်တိုးသွားပါလိမ့်မယ်။

MyWin သုံးပြီး စာရိုက်ဖို့ Panel မှာ ရှိတဲ့ Keyboard Layout Switcher နေရာမှာ သွားရွေးနိုင်သလို Shortcut Key ဖြစ်တဲ့ Super + Space နဲ့လည်း ပြောင်းလဲရွေးချယ် အသုံးပြုနိုင်ပါတယ်။

Myanmar Unicode Keyboards for Ubuntu

August 6 2016
unicode keyboard input keyman kmfl ubuntu

အလျှင်တုန်းကတော့ Unicode နဲ့ Zawgyi အတွက် mm-kb ဆိုတာလုပ်ပေးထားဖူးပါတယ်။ ရိုးရိုး တမ်းတမ်း logical order အတိုင်းရိုက်ရတဲ့ Keyboard ပါ။ MyWin အကြောင်းရေးပေးပြီးနောက် တချို့က Myanmar 3 Standard Layout အတိုင်းရိုက်လို့ရတာမရှိဘူးလားမေးလာကြတဲ့အတွက် ရှိသမျှ အကုန်စုစည်းပြီး Installer လေးလုပ်ပေးဖို့စိတ်ကူးမိသွားတာပါ။ ဒါပေမယ့် ThanLwinSoft ကလုပ်ပေးထားတဲ့ထဲမှာ မြန်မာ၊ ပအိုဝ်းနဲ့ စကောကရင် ပဲရှိသေးပြီး ရှမ်းအတွက်မပါသေးပါဘူး။ ဒါကြောင့် ရှမ်းအတွက်ပါထပ်ဖြည့်ပြီး Installer Package လေးထပ်လုပ်ပေးလိုက်ပါတယ်။

Installation ကတော့အောက်ပါအတိုင်းဖြစ်ပါတယ်။ ပေးထားတဲ့ Command တွေကို Terminal မှာ တစ်ကြောင်းချင်းရိုက်ထည့်ပေးလိုက်ပါ။

Internet ချိပ်ဆက်ထားဖို့တော့လိုအပ်ပါတယ်။

$ wget https://github.com/naingyeminn/mm-kb-kmfl/archive/master.zip -O mm-kb-kmfl.zip
$ unzip mm-kb-kmfl.zip
$ cd mm-kb-kmfl-master
$ make install

ပြီးလျှင်တော့ ပုံမှာပြထားတဲ့အတိုင်း Text Entry Settings… ထဲဝင်လိုက်ပါ။

”+” button ကိုနှိပ်ပြီး Input Source list ထဲ “other” လို့ရိုက်ပြီးရှာလိုက်ပါ။ ပေါ်လာတဲ့ Input Source တွေထဲကမှ ကြိုက်နှစ်သက်ရာကို တစ်ခုချင်းရွေးပြီး Add လုပ်လိုက်ပါ။

Input sources to use ဆိုတဲ့ list ထဲမှာတိုးသွားပါလိမ့်မယ်။

အခုဆိုရင်ကြိုက်နှစ်သက်ရာ Keyboard ကို ရွေးချယ်ပြီးအသုံးပြုနိုင်ပါပြီ။ Panel မှာ ရှိတဲ့ Keyboard Layout Switcher နေရာမှာ သွားရွေးနိုင်သလို Shortcut Key ဖြစ်တဲ့ Super + Space နဲ့လည်း ပြောင်းလဲရွေးချယ် အသုံးပြုနိုင်ပါတယ်။

လိုအပ်ချက်များရှိပါက ဆက်သွယ် အကြံပြုနိုင်ပါတယ်။

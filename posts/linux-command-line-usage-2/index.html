Linux Terminal / Command Line Tricks (2)

August 4 2015
ubuntu linux terminal cli tips&tricks

Linux/Unix ရဲ့ Bash Shell Terminal မှာ command တွေကို မြန်မြန်ဆန်ဆန် Edit လုပ်နိုင်စေမယ့် Shortcut လေးတွေပါ။ ရိုက်လိုက်တဲ့ command တစ်ကြောင်းမှာ အမှားပါနေလို့ပြန်ပြင်မယ်ဆိုရင် ကိုယ်တွေလုပ်နေကျ သမရိုးကျနည်းလမ်းအတိုင်း Arrow key တွေနဲ့ ရှေ့နောက်ရွှေ့ Delete, Backspace တို့နဲ့ပြန်ဖျက်၊ ပြန်ရေးနဲ့ အချိန်ကြာပါတယ်။ အခုထုတ်နှုတ်ပေးထားတဲ့ Shortcut လေးတွေသိထားမယ်ဆိုရင်တော့ command တွေကို မြန်မြန်ဆန်ဆန်ရိုက်နိုင် ပြင်နိုင်မှာဖြစ်တဲ့အတွက် အလုပ်ပိုတွင်ပါလိမ့်မယ်။ Linux/Unix Admin တွေအနေနဲ့ မဖြစ်မနေ သိထားသင့်တဲ့ Shortcut လေးတွေပဲဖြစ်ပါတယ်။

Command Editing Shortcuts

  • Control + a : Command အစသို့

  • Control + e : Command အဆုံးသို့

  • Control + k : Cursor နေရာမှစ၍ Command အဆုံးထိဖျက်

  • Control + u : Cursor နေရာမှစ၍ Command အစထိဖျက်

  • Control + w : Cursor နေရာမှစ၍ စာလုံး(Word) အစထိဖျက်

  • Control + y : လက်တလောဖျက်ထားသော စာလုံး (သို့) Command အား ပြန်ရယူ

  • Control + xx : Cursor ရှိရာနေရာနှင့် Command အစကြား အပြန်အလှန်သွား

  • Alter + b : စာလုံးတစ်လုံးစာ နောက်ပြန်ရွှေ့

  • Alter + f : စာလုံးတစ်လုံးစာ အရှေ့သို့ရွှေ့

  • Alter + d : Cursor နေရာမှစ၍ စာလုံး(Word) အဆုံးထိဖျက်

  • Alter + c : စာလုံး၏ပထမအက္ခရာ(Character) အား (သို့) Cursor ရှိရာအက္ခရာအား Capital Letter သို့ပြောင်း

  • Alter + u : Cursor နေရာမှစ၍ စာလုံးအဆုံးထိ Uppercase သို့ပြောင်း

  • Alter + l : Cursor နေရာမှစ၍ စာလုံးအဆုံးထိ Lowercase သို့ပြောင်း

  • Alter + t : Cursor ရှိရာနေရာမှစာလုံး နှင့် ရှေ့စာလုံးနေရာလဲ

  • Control + f : အက္ခရာ(Character) တစ်လုံးစာ အရှေ့သို့ရွှေ့

  • Control + b : အက္ခရာတစ်လုံးစာ အနောက်သို့ရွှေ့

  • Control + d : Cursor ရှိရာနေရာမှ အက္ခရာကိုဖျက်

  • Control + h : Cursor ရှေ့မှ အက္ခရာကိုဖျက်

  • Control + t : Cursor ရှိရာနေရာမှအက္ခရာ နှင့် ရှေ့အက္ခရာနေရာလဲ

Command Recall & Control Shortcuts

  • Control + p : Command များ နောက်ပြန်ရှာ

  • Control + n : Command များ ရှေ့တိုးရှာ

  • Alter + . : နောက်ဆုံးရိုက်ထားသော Command ၏နောက်ဆုံးစာလုံးအားပြန်ရယူ

  • Control + l : Terminal Screen ရှင်း

  • Control + c : Run နေသော Command အားအဆုံးသတ်

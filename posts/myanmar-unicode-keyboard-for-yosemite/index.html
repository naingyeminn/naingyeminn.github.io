Myanmar Unicode Keyboard for Mac OS X (Yosemite)

December 5 2014
unicode keyboard myanmar3 mac osx yosemite

Mac OS X Yosemite စသုံးတော့ Unicode အတွက် keyboard ရှာဖြစ်တယ်။ ပါလာတဲ့ Default မြန်မာစာ လက်ကွက်က ပုံမှန်ရိုက်နေကျ မြန်မာ unicode keyboard layout အတိုင်မဟုတ်တော့ ရိုက်ရတာအဆင်မပြေဘူး။ ဒါနဲ့ KeyMagic ပဲသုံးမယ်လေဆိုပြီး install လုပ်လိုက်တယ်။ စာရိုက်ရတာတော့အဆင်ပြေပါတယ်။ ဒါပေမယ့် မြန်မာလိုရိုက်နေရာကနေ ခဏနားလိုက်လို့ Screen Lock ကျသွားရင် ပြန်ဝင်ဖို့ password ရိုက်မရတော့ဘူး။ Guest Account ထဲဝင်ပြီးပြန်ထွက်မှ password ပြန်ရိုက်လို့ရတယ်။ KeyMagic ရေးတဲ့ သန့်သက် ကိုမေးကြည့်တော့ အဲဒါက Yosemite ရဲ့ Bug ဆိုတော့ ဘာမှလုပ်ပေးလို့မရပါဘူးတဲ့။ အမြဲလိုလိုအဲ့လိုကြီးဝင်နေရတာ သိပ်မဟန်တာကြောင့် Keyboard Input Source ဘယ်လိုလုပ်ရမလဲဆိုပြီး ရှာကြည့်ဖြစ်တော့ လွယ်လွယ်ကူကူပါပဲ အဖြေတန်းတွေ့တယ်။ Ukelele ဆိုတဲ့ software လေးပါ။ အဲဒါလေးနဲ့ ရှိနေတဲ့ Keyboard Layout တွေကို စိတ်ကြိုက် ပြင်လို့ရတယ်ဆိုတော့ Default Myanmar Keyboard ကို Myanmar3 Layout အတိုင်းဖြစ်အောင် ပြန်ပြီးပြင်လိုက်တယ်။ ရိုက်နေကျအတိုင်းဖြစ်သွားတော့ တော်တော်လေးအဆင်ပြေသွားတယ်။ KeyMagic မှာလို password ရိုက်မရတာမျိုးလဲမရှိတော့ဘူး။ အဲဒါကြောင့် Facebook က Myanmar Unicode Area မှာ တင်ပြီး share ပေးထားလိုက်တယ်။ ဒါပေမယ့် Facebook က ပြန်ရှာရခက်တာကြောင့် ဒီမှာထပ်ရေးပြီး Share ပေးပါ့မယ်။

Installation ကတော့ ခက်ခက်ခဲခဲကြီးမဟုတ်ပါဘူး။ ပေးထားတဲ့ zip ဖိုင်လေးကို download ဆွဲ double click နှိပ်ပြီး extract လုပ်။ ထွက်လာတဲ့ ဖိုင်ကို /Library/Keyboard Layouts ဆိုတဲ့ folder ထဲသွားထည့်ပေးလိုက်ရုံပါပဲ။ ပြီးလျှင်တော့ ပုံမှန်အတိုင်း System Preferences > Keyboard > Input Sources ကနေ ရွေးပြီး သုံးနိုင်ပါပြီ။

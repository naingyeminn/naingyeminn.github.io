iSCSI with Multipath in CentOS7

April 11 2018
linux video tutorial centos7 administration

iSCSI (Internet Small Computer Systems Interface) ဆိုတာ SAN (Storage Area Network) Protocol တစ်ခုပါ။ အလွယ်ပြောရရင် Computer တစ်လုံးမှာရှိတဲ့ Storage Device/Space ကို အခြား Computer တစ်လုံးရဲ့ Block Device/Hard Disk အနေနဲ့ Network ကနေတဆင့် ဆွဲယူချိပ်ဆက်အသုံးပြုတဲ့အခါမှာသုံးပါတယ်။ iSCSI ရဲ့အားသာချက်ကတော့ သာမန် IP Network ပေါ်မှာအလုပ်လုပ်တာဖြစ်ပြီး Fiber Channel လို SAN Switch မလိုအပ်ပါဘူး။ Network ပေါ်ကနေတဆင့် ချိပ်ဆက်အသုံးပြုတာဖြစ်တဲ့အတွက် အကယ်၍များ Network Link ပြတ်တောက်သွားခဲ့ရင် ပြဿနာတက်နိုင်ပါတယ်။ ဒါကြောင့် Network Link ကို တစ်ခုထဲအသုံးပြုတာထက် ၂ခုအနည်းဆုံးထားပြီး ချိပ်ဆက်မှသာ High Availability ဖြစ်မှာပါ။ ဒီလိုချိပ်ဆက်ခြင်းမျိုးကို Multipath လို့ခေါ်ခြင်းဖြစ်ပါတယ်။ အခု ဒီ Video လေးမှာတော့ CentOS 7 ပေါ်မှာ iSCSI Drive ကို Multipath သုံးပြီး ဘယ်လိုချိပ်ဆက်မလဲဆိုတာကို ပြသပေးထားပါတယ်။
